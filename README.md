# Shipmate Logistics — Client Website & Content Guide

Welcome to the official website repository for **Shipmate Logistics**. This guide is prepared to help you easily manage, update, and maintain your company website.

---

## 🌟 Website Overview

Your website is built with a fast, modern design tailored for logistics and supply chain management:

- **Pan-India Logistics Branding**: Pristine white truck fleet photography, modern typography, and dark navy glassmorphism styling.
- **Clean URLs**: Clean, professional address bar links (`/about`, `/services`, `/fleet`, `/contact`).
- **Mobile Web App (PWA)**: Built-in mobile installation prompt for iPhone and Android users.
- **Fast Performance**: Optimized loading times without heavy dependencies.

---

## ✏️ Managing Website Content & Media

You have two easy ways to update text, photos, and videos on your website:

### Option A: Content Manager Portal (No Coding Required)
Visit your website's admin portal:
👉 **`https://shipmatelogistics.in/admin/`**

1. Log in with your email and password.
2. Select any section (**Hero Video**, **Services**, **Fleet Vehicles**, **About Story**, or **Contact Details**).
3. Update text, upload new photos or videos, and click **Publish**.
4. Your website will update automatically!

---

### Option B: Quick Media Replacement (File Folder Method)

All images and videos are stored inside the `assets/` folder:

| Section | Location | Recommended Size |
| :--- | :--- | :--- |
| **Hero Background Video** | `assets/videos/hero-loop.mp4` | 1920x1080 (16:9) |
| **Hero Video Poster** | `assets/images/hero-poster.webp` | 1920x1080 (16:9) |
| **Company Logo** | `assets/images/logos/mainlogo.webp` | 512x512 |
| **About Story Photo** | `assets/images/about-story.webp` | 1200x900 (4:3) |
| **Service Cards** | `assets/images/services/` | 800x600 (4:3) |
| **Fleet Vehicles** | `assets/images/fleet/` | 800x600 (4:3) |
| **Leadership Photos** | `assets/images/team/` | 400x400 (1:1) |

To replace any photo, simply upload your new image into the respective folder using the **exact same file name**.
